Recovering Jupyter Notebook from Command History

Date Sat 27 June 2020 Tags Python / Jupyter Notebook

Jupyter notebook recommends to use tools such as Git and Subversion for version control. However, it may not seem necessary to keep track of all the changes we make when experimenting with Jupyter notebook. In that case, if we ever want to check the commands we ran beyond the ability of the undo command, we can check the command history using %history.

%history  # Show the command history of the current IPython session

If the above command is not enough, the following command can be used which will show the entire command history including the very first command had been written ages ago.

%history -g  # Show the entire command history
